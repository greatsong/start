INTRO · 왜 데이터인가

AI는 '열매'를 다루는 기술입니다

번역, 요약, 그림, 코드… 화려한 열매는 AI가 맺어줍니다. 그런데 그 열매를 맺게 하는 뿌리는 무엇일까요?

🌳 AI가 열매라면, 그 뿌리는 데이터입니다. 좋은 데이터를 만지고 읽을 줄 아는 사람이, AI 시대에도 진짜 힘을 가집니다.
HOOK · 질문으로 시작

데이터는 이런 질문에 답합니다

아래 질문들, 감으로 답하지 말고 데이터로 확인해 볼까요?

우리 학교 동네엔 몇 살이 많을까?
→ 인구 피라미드로 한눈에
청소년이 유독 많다면… 학군지?
→ 만 14세 비율로 확인
전국에서 만 14세가 제일 많은 동네는?
→ 1위 대치1동 · 2위 범어4동…
대치1동의 '쌍둥이 동네'는?
→ 평촌·노원·둔산·수성… 전부 학군지!

데이터가 학군지를 알아봅니다. 직접 우리 동네를 넣어 찾아보고, 인사이트를 나눠 주세요!

API · 데이터를 받아오는 곳

'API'라는 창구가 있습니다

API는 데이터나 기능을 받아올 수 있는 창구예요. 우리가 직접 모으지 않아도, 이미 만들어진 데이터를 불러다 쓸 수 있습니다.

🆓 무료 API

공공데이터포털, 나이스(NEIS) 급식·학교 정보 등. 인증키가 없어도 되는 것도 많아요.

💳 유료 API

지도·결제·고성능 AI 등. 트래픽/기능에 따라 요금이 붙습니다.

오늘은 NEIS(나이스) API로 우리 학교의 오늘 급식을 인증키 없이 실시간으로 불러옵니다.
